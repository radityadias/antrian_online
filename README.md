# antrian_online
Laravel

**Cara pakai**

1. Jalankan perintah *copy .env.example .env* pada terminal
2. Jalankan perintah *composer install*
3. Jalankan perintah *php artisan key:generate:
4. Atur koneksi database yang ada di file *.env*
5. Jalankan perintah *php artisan migrate*
6. Jalankan perintah *php artisan db:seed --path=\database\seeders\admin.php*
7. Jalankan perintah *npm install*
8. Buat terminal baru dan jalankan perintah *php artisan serve*
9. Buat terminal baru dan jalankan perintah *npm run dev*
10. Masuk ke local server yang tertera pada terminal *php artisan serve*
